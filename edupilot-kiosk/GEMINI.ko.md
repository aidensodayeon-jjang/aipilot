# D-LAB 스마트 출결 시스템 V3

## 프로젝트 개요

이 프로젝트는 D-LAB 코딩 학원의 출결을 관리하기 위한 웹 애플리케이션인 D-LAB 스마트 출결 시스템 V3입니다. 학생들이 출석 체크를 할 수 있는 키오스크 모드, 출결, 일정, 학생 및 직원을 관리할 수 있는 관리자 대시보드, 그리고 강력한 인증 및 보안 기능을 갖추고 있습니다.

**주요 기능:**
- **인증 및 보안:** Supabase 인증, 관리자 로그인, 직원 관리, 행 수준 보안(RLS), 역할 기반 접근 제어(마스터 관리자/스태프).
- **키오스크 모드:** 전화번호를 통한 학생 출석 체크, 시간표 기반 자동 출석, 수업 정보가 포함된 성공 화면, 마케팅 배너.
- **관리자 대시보드:** 실시간 출결 현황, 시간표 그리드 뷰, 학생 관리, 알림 설정, 직원 관리.

## 기술 스택

- **프레임워크:** Next.js 16 (앱 라우터)
- **스타일링:** Tailwind CSS v4
- **상태 관리:** Zustand
- **아이콘:** Lucide React
- **데이터베이스:** Supabase (PostgreSQL)
- **인증:** Supabase Auth

## 빌드 및 실행

### 1. 의존성 설치
```bash
npm install
```

### 2. Supabase 설정

#### 2.1 환경 변수
`env.example`을 `.env.local`로 복사하고 Supabase 프로젝트 정보로 업데이트합니다:
```bash
cp env.example .env.local
```
`.env.local` 편집:
```env
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
```

#### 2.2 데이터베이스 스키마 적용
1. Supabase 대시보드 → SQL 편집기로 이동합니다.
2. `SUPABASE_SCHEMA.sql`의 내용을 복사하여 실행합니다.

#### 2.3 마스터 관리자 계정 생성
자세한 내용은 `SETUP_AUTH.md`를 참조하십시오.
**요약:**
1. Supabase 대시보드 → 인증 → 사용자 → "사용자 추가".
2. 이메일: `admin@dlab.com`, 비밀번호: `pass1234`.
3. "자동 사용자 확인"을 선택합니다.
4. `CREATE_MASTER_ADMIN.sql` 스크립트를 실행합니다.

### 3. 개발 서버 실행
```bash
npm run dev
```

## 개발 컨벤션

- **스타일링:** 스타일링에는 Tailwind CSS v4가 사용됩니다.
- **상태 관리:** 상태 관리에는 Zustand가 사용됩니다.
- **아이콘:** Lucide React가 아이콘을 제공합니다.
- **언어:** 애플리케이션은 한국어로 완전히 현지화되었습니다.

## 보안 기능

- **행 수준 보안(RLS):** 학생, 수업, 등록(인증된 사용자만 해당), 출석 로그(키오스크 삽입, 인증된 보기) 및 프로필(관리자 생성/수정/삭제)에 대해 구현되었습니다.
- **역할 시스템:**
    - **마스터 관리자:** 직원 관리를 포함한 모든 권한.
    - **스태프:** 데이터 보기 및 수정 권한.

## UI/UX 기능

- 완전한 한국어 지원.
- D-LAB 브랜드 색상 준수(파란색: #1E40AF, 보라색: #7C3AED).
- 반응형 디자인.
- 그라데이션 및 애니메이션 효과.
- 프리미엄 디자인 미학.

## 문서

- `SETUP_AUTH.md`: 인증 시스템 설정 가이드.
- `SUPABASE_SCHEMA.sql`: 데이터베이스 스키마 정의.
- `CREATE_MASTER_ADMIN.sql`: 마스터 관리자 계정 생성을 위한 스크립트.
