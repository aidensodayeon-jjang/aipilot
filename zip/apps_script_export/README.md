# Google Apps Script 배포 가이드

이 대시보드를 Google Sheets에 연동하여 사용하려면 다음 단계를 따르세요.

## 1. Google Sheets 준비
1. 데이터가 있는 구글 스프레드시트를 엽니다.
2. 상단 메뉴에서 `확장 프로그램` > `Apps Script`를 클릭합니다.

## 2. 코드 복사 및 붙여넣기
Apps Script 에디터가 열리면 다음 파일들을 생성하고 코드를 붙여넣으세요.

### `Code.gs`
1. 기본으로 있는 `Code.gs` 파일의 내용을 모두 지웁니다.
2. 이 폴더에 있는 `Code.gs` 파일의 내용을 복사해서 붙여넣습니다.
3. 저장합니다 (Ctrl+S).

### `index.html`
1. `파일 +` 버튼을 누르고 `HTML`을 선택합니다.
2. 파일 이름을 `index`로 지정합니다 (확장자 `.html`은 자동 생성됨).
3. 이 폴더에 있는 `index.html` 파일의 내용을 복사해서 붙여넣습니다.
4. 저장합니다.

## 3. 테스트 및 배포
1. 상단 툴바에서 `배포` > `새 배포`를 클릭합니다.
2. '유형 선택' 톱니바퀴를 눌러 `웹 앱`을 선택합니다.
3. 다음 설정을 확인합니다:
   - **설명**: (선택사항)
   - **다음 사용자 계정으로 앱 실행**: `나(Me)`
   - **액세스 권한이 있는 사용자**: `Google 계정이 있는 모든 사용자` 또는 `나` (용도에 맞게 선택)
4. `배포` 버튼을 클릭합니다.
5. **권한 검토** 팝업이 뜨면 계정을 선택하고 `고급` > `(프로젝트 이름)으로 이동(안전하지 않음)`을 클릭하여 권한을 허용합니다.
6. 제공된 **웹 앱 URL**을 클릭하여 대시보드를 확인합니다.

## 주의사항
- 스프레드시트의 첫 번째 시트 데이터를 자동으로 읽어옵니다.
- 데이터의 헤더(첫 번째 행) 이름이 `비고`, `수강료결제`, `결제금액`, `학생이름`, `학교`, `학년`, `수강과목`, `수강시간`, `담당` 등 코드에 정의된 것과 일치해야 합니다.
